# Commit Message 格式

- Feat: 新增/修改功能 (feature)。
- Fix: 修補 bug (bug fix)。
- Docs: 文件 (documentation)。例：新增圖片、修改文字檔
- Style: 格式 (不影響程式碼運行的變動)。例：更動 html scss
- Refactor: 重構 (既不是新增功能，也不是修補 bug 的程式碼變動)。
